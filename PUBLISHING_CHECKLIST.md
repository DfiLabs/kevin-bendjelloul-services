## Publishing checklist (KVB Énergie)

### Contact
- [ ] Phone number is correct (tap-to-call works on mobile)
- [ ] WhatsApp link works and opens the right chat + prefilled message
- [ ] Email destination is correct (form mailto opens with correct `to:`)

### Services
- [ ] Examples (3 bullets) match real offerings
- [ ] “Délais typiques” statements are accurate
- [ ] “À partir de” / pricing wording is accurate (or remove it)

### Legal / compliance
- [ ] Mentions légales are correct and not boilerplate
- [ ] Privacy policy matches actual data usage
- [ ] Cookie policy matches actual analytics/cookies (add consent banner if needed)
- [ ] APE/activities are consistent with the page positioning (electrician vs cleaning)

### UX / QA
- [ ] CTAs work: “Demander un devis”, “Appeler”, “WhatsApp”
- [ ] Mobile view tested (iPhone + Android)
- [ ] Form fields usable and readable
- [ ] Page load ok on slow connection

### SEO / structured data
- [ ] JSON-LD type is correct (Electrician/LocalBusiness)
- [ ] Service area is correct (Montpellier + Hérault)
- [ ] FAQ content matches FAQPage markup
- [ ] No placeholder emails remain in UI
